Code 4000 Test Driven Development Workshop
===
This simple project is a starter template for demonstrating TDD using NodeJS with Mocha and Chai.

Prerequisites
---
npm 5+
node 8+


To build
---
```
npm install
```

To run the tests
---
```
npm test
```
